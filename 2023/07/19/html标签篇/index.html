常用的html标签

发表于2023-07-19|更新于2023-07-19|IT类
|字数总计:1.3k|阅读时长:3分钟

第一部分html标签

  1. html：定义 HTML 文档的根元素。
  2. head：定义 HTML 文档的头部，包含元数据信息。
  3. ol：定义文档的标题，显示在浏览器的标题栏或标签页上。
  4. body：定义 HTML 文档的主体部分，包含可见内容。
  5. h1 - h6：定义标题，从 1 到 6 级别，h1 为最高级别。
  6. p：定义段落，用于组织文本内容。
  7. a：定义超链接，用于创建指向其他网页、文件、位置或电子邮件地址的链接。
  8. img：定义图像，用于在页面中插入图片。
  9. div：定义一个块级容器，用于组合其他 HTML 元素，并用 CSS 进行样式化和布局控制。
  10. span：定义内联容器，用于对文本或行内元素进行样式化。
  11. ul：定义无序列表，以项目符号显示项目。
  12. ol：定义有序列表，以数字或字母表示每个项目。
  13. li：定义列表项，在无序或有序列表中表示每个项目。
  14. table：定义表格，用于显示多行多列的数据。
  15. tr：定义表格的行。
  16. td：定义表格的单元格。
  17. th：定义表格的表头单元格。
  18. form：定义用于用户输入的表单。
  19. input：定义输入字段，用于接收用户输入。
  20. button：定义按钮，用于触发特定的操作或提交表单。
  21. select：定义下拉列表框，供用户选择其中一个选项。
  22. textarea：定义多行文本输入框，用于接收多行文本输入。
  23. label：定义表单控件的标签。
  24. fieldset：定义一组相关的表单元素。
  25. legend：定义 fieldset 元素的标题。
  26. iframe：定义内联框架，用于在当前页面中嵌入其他页面。
  27. audio：定义音频播放器。
  28. video：定义视频播放器。
  29. script：定义客户端脚本，通常是 JavaScript。
  30. style：定义内部样式表。
  31. link：定义外部样式表或引入其他文件。
  32. meta：定义关于 HTML 文档的元数据信息。

第二部分html标签

  1. header：定义文档或区域的页眉。
  2. nav：定义导航链接的容器。
  3. main：定义文档的主要内容，每个文档只能有一个元素。
  4. article：定义独立的文章内容。
  5. section：定义文档中的节或区域。
  6. aside：定义页面的侧边栏内容。
  7. footer：定义文档或区域的页脚。
  8. figure：定义嵌入的图像、图表、照片等单独的内容块。
  9. figcaption：定义 元素的标题或说明性文本。
  10. blockquote：定义长引用，通常包含在中。
  11. cite：定义引用的作品的标题。
  12. abbr：定义缩写或首字母缩写。
  13. code：定义计算机代码片段。
  14. pre：定义预格式化的文本，保留空格和换行符的原始格式。
  15. mark：定义突出显示的文本。
  16. strong：定义重要的文本。
  17. em：定义强调的文本。
  18. sub：定义下标文本。
  19. sup：定义上标文本。
  20. hr：定义水平分隔线。
  21. br：定义换行符。
  22. progress：定义进度条。
  23. meter：定义度量衡，例如温度或文件下载进度。
  24. datalist：定义输入字段的选项列表。
  25. optgroup：定义选择列表中的选项组。
  26. option：定义选择列表中的选项。
  27. iframe：定义嵌入的可视化窗口。
  28. map：定义图像地图。
  29. area：定义图像地图中的可点击区域。
  30. canvas：定义绘图区域，用于通过 JavaScript 绘制图形。
  31. svg定义可缩放矢量图形。
  32. time：定义日期或时间。
  33. details：定义用户可以查看或隐藏的额外详细信息。
  34. summary：定义 元素的摘要或标题。

第三部分html标签

  1. address：定义联系信息或作者/拥有者的联系地址。
  2. b：定义粗体文本。
  3. big：定义大号文本。
  4. center：定义居中对齐的内容（已被废弃，请使用 CSS 进行样式化）。
  5. del：定义删除的文本。
  6. dfn：定义术语的定义。
  7. font：定义文本的字体、颜色和大小（已被废弃，请使用 CSS 进行样式化）。
  8. ins：定义插入的文本。
  9. kbd：定义键盘文本。
  10. meter：定义度量衡，例如温度或文件下载进度。
  11. output：定义计算的结果。
  12. q：定义短引用。
  13. rp：在不支持 ruby 注释显示时提供回退内容。
  14. rt：定义 ruby 注释的解释。
  15. ruby：定义 ruby 注释（用于东亚字符注音或字符解释）。
  16. s：定义不准确或不相关的文本。
  17. samp：定义计算机程序或脚本的示例输出。
  18. small：定义小号文本。
  19. strike：定义删除的文本（已被废弃，请使用del标签)。
  20. strong：定义重要的文本。
  21. sub：定义下标文本。
  22. sup：定义上标文本。
  23. tt：定义打字机或等宽文本。
  24. u：定义带有下划线的文本。
  25. var：定义变量。
    26. 这些标签可能不常用或已被废弃，因此在编写 HTML 页面时，应优先考虑使用语义化的标签，并结合 CSS 进行样式化和布局控制。
